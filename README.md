# jellyfish-eye

TensorFlow sample. Classifying depth sensor data.

## How to use

```bash
# Please setup python 3.x and TensorFlow 1.x

$ python -m jellyfish_eye.train  # for training
$ python -m jellyfish_eye.serve  # for serving
```
